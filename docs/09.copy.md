# 画面文言一覧（UIコピー）

## 一覧
- タイトル: 記録一覧
- 管理者ログイン: 管理者ログイン
- 管理者の記録追加ボタン: 記録追加
- 空状態: 記録がありません。最初の記録を追加しましょう
- 詳細ボタン: 詳細を見る
- ラベル: 合計セット数 / 有酸素合計時間 / 有酸素合計距離
- ラベル: 推定消費カロリー

## 詳細
- タイトル: 記録詳細
- 戻るリンク: 一覧へ戻る
- 体調メモ空表示: 体調メモなし
- ラベル: 種目 / 部位 / セット / 回数 / 重量 / 種別 / 時間 / 距離
- ラベル: 推定消費カロリー

## 記録追加
- タイトル: 記録追加
- 戻るリンク: 一覧へ戻る
- セクション見出し: 筋トレ / 有酸素 / 体調メモ
- 筋トレ行追加: 追加
- 筋トレ行削除: 削除
- 保存ボタン: 保存
- 入力ラベル: 日付 / 部位 / 種目名 / セット数 / 回数 / 重量 / 種別 / 時間 / 距離 / 体調メモ
- 通知（同日保存エラー）: 同じ日付の記録が既に存在します。
- ラベル: 推定消費カロリー

## 管理者メニュー
- タイトル: 管理者メニュー
- メニュー項目: 記録一覧 / 記録追加 / プロフィール / マスター管理 / データ出力 / ログアウト

## プロフィール
- タイトル: プロフィール
- 体重ラベル: 体重 (kg)
- 保存ボタン: 保存

## マスター管理
- タイトル: マスター管理
- タブ: 部位 / 種目 / 有酸素種別
- 追加入力プレースホルダー: 新しい項目を追加
- 追加ボタン: 追加
- 削除確認: この項目を削除します。よろしいですか？

## データ出力
- タイトル: データ出力
- 戻るリンク: 戻る
- 期間ラベル: 開始日 / 終了日
- 形式選択: CSV / JSON
- 出力ボタン: エクスポート
