# 画面フロー（遷移）

## 一般ユーザー
- 一覧 → 詳細（「詳細を見る」）
- 詳細 → 一覧（「一覧へ戻る」）

## 管理者
- 一覧 → 管理者ログイン（ヘッダー右上リンク）
- 管理者ログイン → 一覧（ログイン成功時）
- 一覧 → 記録追加（一覧上部「記録追加」）
- 記録追加 → 一覧（保存後）
- 管理者メニュー → 記録一覧(管理操作)
- 管理者メニュー → 記録追加
- 管理者メニュー → プロフィール（体重設定）
- 管理者メニュー → マスター管理
- 管理者メニュー → データ出力
- 管理者メニュー → ログアウト
- データ出力 → 前画面（「戻る」）

## 制御
- 未ログインで管理者URLにアクセスした場合はログイン画面へリダイレクト
- 管理者メニューはログイン後のみ表示（一般ユーザー非表示）
- ログアウトは画面右上固定表示
