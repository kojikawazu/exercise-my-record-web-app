# カロリー計算（目安）

## 方針
- 目安（概算）で算出する
- 体重はプロフィール（Supabase）に保存する
- 1日合計 / 筋トレ合計 / 有酸素合計 を表示する

## 入力
- 体重（kg）
- 有酸素: 時間（分）
- 筋トレ: 時間を別途入力しないため、固定係数で概算する

## 計算式（暫定）
### 有酸素（時間ベース）
- `消費カロリー = METs × 体重(kg) × 時間(時間)`
- METs は種別ごとの固定値を使用（暫定）
  - ラン: 8.0
  - ウォーク: 4.0

### 筋トレ（概算）
- `消費カロリー = 体重(kg) × 0.1 × セット数合計`
- セット数合計はその日の合計セット数

## 表示
- 一覧/詳細/記録追加の全てに表示する
- 一覧はカード下部に表示
- 詳細は上部に表示
- 記録追加は画面下部に表示
- 表示ラベルは「推定消費カロリー」
