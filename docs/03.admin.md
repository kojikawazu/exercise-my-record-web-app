# 管理者・権限 要件

## ロール
- 管理者: 記録の追加/編集/削除、マスター管理、データ出力
- 一般ユーザー: 閲覧のみ（一覧/詳細）

## 管理者機能
- 記録管理: 追加 / 編集 / 削除
- マスター管理: 部位、種目名、有酸素種別（ラン/ウォーク）の編集
- データ出力: CSV/JSON エクスポート（Issue #20 で削除予定）

## 認証
- 管理者ログイン必須
- 認証方式: Google OAuth
- 一般ユーザーはログイン不要

## 画面スコープ
- 管理者のみ: 記録追加、記録編集/削除操作、マスター管理、データ出力（Issue #20 で削除予定）
- 一般ユーザー: 一覧、詳細、カレンダー、推移グラフ（閲覧のみ）

## 編集導線
- 主導線: 一覧/詳細画面から直接編集画面へ遷移
- 補助導線: 管理者メニューから記録一覧/記録追加へ遷移
