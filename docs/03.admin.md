# 管理者・権限 要件

## ロール
- 管理者: 記録の追加/編集/削除、マスター管理、データ出力
- 一般ユーザー: 閲覧のみ（一覧/詳細）

## 管理者機能
- 記録管理: 追加 / 編集 / 削除
- マスター管理: 部位、種目名、有酸素種別（ラン/ウォーク）の編集
- データ収集/出力: 記録のエクスポート（CSV/JSON は要検討）

## 認証
- 管理者ログイン必須
- 認証方式: Google OAuth
- 一般ユーザーはログイン不要

## 画面スコープ
- 管理者のみ: 記録追加、編集/削除操作、マスター管理、エクスポート
- 一般ユーザー: 一覧、詳細（閲覧のみ）
