# 画面アウトライン（レイアウト入力）

## 1. 記録追加
- 日付ピッカー
- 体調メモ（任意）
- 筋トレ行（複数追加）
  - 部位（セレクト）
  - 種目名（テキスト）
  - セット数（数値）
  - 回数（数値）
  - 重量（数値）
- 有酸素ブロック（任意）
  - 種別（ラン/ウォーク）
  - 時間（数値）
  - 距離（数値）
- 保存ボタン

## 2. 一覧
- 日付リスト
- 1日カード
  - 合計セット数
  - 有酸素の合計時間
  - 有酸素の合計距離
- 詳細へのリンク/ボタン

## 3. 詳細
- 日付
- 体調メモ
- 筋トレ一覧
  - 部位 / 種目 / セット / 回数 / 重量
- 有酸素詳細（任意）
  - 種別 / 時間 / 距離

## 共通UI部品
- 日付ピッカー
- 数値入力
- セレクト（部位、ラン/ウォーク）
- 追加/削除ボタン
- バリデーション表示
- 保存ボタン
- 空状態表示
