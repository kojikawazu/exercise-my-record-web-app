# 仕様（確定）

## 目的
フィットネス記録MVPの仕様を明文化し、実装判断を固定する。

## スコープ
- 1日1レコード（同日保存はエラーで弾く）
- 1日に複数の筋トレ種目を追加可能
- 有酸素は任意

## ロール/権限
- 管理者: 記録の追加/編集/削除、マスター管理、データ出力
- 一般ユーザー: 一覧/詳細の閲覧のみ

## 認証/セッション
- 認証はGoogle OAuthをMVPに含める
- 管理者ログイン導線は一覧画面に表示する
- 未ログインで管理者URLにアクセスした場合はログイン画面へリダイレクト
- ログイン後は一覧画面へ遷移する
- セッションはブラウザを閉じても保持する

## 画面
- 記録追加
- 一覧
- 詳細
- 管理者メニュー
- プロフィール（体重設定）
- マスター管理
- データ出力

## データ仕様
### 筋トレ
- 部位（セレクト）
- 種目名（テキスト）
- セット数（数値）
- 回数（数値）
- 重量（数値）

### 有酸素
- 種別（ラン/ウォーク）
- 時間（数値）
- 距離（数値）

### 任意
- 体調メモ（500文字まで、改行可）

## 入力/バリデーション
- 数値項目: セット数、回数、重量、時間、距離
- 数値項目は小数入力を許可する
- 有酸素種別は「ラン/ウォーク」
- 1日に複数の筋トレ項目を追加可能
- バリデーション表示は必須（ただし保存を抑止しない）
- 必須の表示（必須ラベル/アスタリスク等）は行わない
- 同日保存時はエラーとして弾き、通知UIを表示する

## 単位
- 重量=kg
- 距離=km
- 時間=分

## 一覧
- 並び順は日付の新しい順（降順）
- 1日1カード表示
- 表示項目: 日付/合計セット数/有酸素合計時間/有酸素合計距離/詳細ボタン
- 空状態文言: 「記録がありません。最初の記録を追加しましょう」
- 管理者は一覧上部に「記録追加」ボタンを表示する
- 合計セット数は筋トレ行のセット数合計
- ヘッダーはタイトル＋管理者ログインリンク
- 一覧カード内の並び順は日付→合計セット数→有酸素合計時間→有酸素合計距離→詳細ボタン
- 一覧カードの合計値は単位付きで表示する（例: 12セット/60分/5.2km）
- 一覧の詳細ボタン文言は「詳細を見る」
- 管理者ログインリンクはヘッダー右上に表示する
- 消費カロリー（目安）をカード下部に表示する

## 詳細
- 並び順: 日付→筋トレ一覧→有酸素詳細→体調メモ
- 筋トレ一覧はカード/リスト形式で表示する
- 有酸素詳細は1行表示（種別/時間/距離）
- 体調メモが空の場合は「体調メモなし」を表示する
- ヘッダーはタイトル＋戻るリンク
- 戻るリンク文言は「一覧へ戻る」
- 筋トレ一覧の表示順は種目→部位→セット→回数→重量
- 筋トレ一覧はラベル付きで表示する
- 有酸素詳細はラベル付きで表示する
- 詳細の数値表示は単位付きで表示する
- 消費カロリー（目安）を上部に表示する

## 記録追加
- 日付初期値は空（ユーザーが選択）
- 筋トレ行は初期表示で1行用意する
- 筋トレ行は最少1行を維持する（全削除不可）
- 体調メモの入力UIはテキストエリア
- 有酸素ブロックの初期値は種別=ラン、時間/距離は空
- 保存後は一覧へ戻る
- ヘッダーはタイトル＋戻るリンク
- 戻るリンク文言は「一覧へ戻る」
- 筋トレ行の追加ボタンはセクション末尾に配置する
- 筋トレ行の削除ボタンは各行の右端に配置する
- 入力ラベルを表示する
- 日付ピッカーはカレンダーUIを使用し、年へのジャンプ操作を用意する
- 日付ピッカーはモダンなUIコンポーネントを採用する
- 消費カロリー（目安）を画面下部に表示する

## 管理者メニュー
- 管理者はログイン後に管理者メニューを表示（一般ユーザーには非表示）
- 表示形式はカード/ボタンのグリッド
- 並び順: 記録一覧(管理操作)→記録追加→マスター管理→データ出力→ログアウト
- ログアウト導線は画面右上に固定表示する
- 管理者メニューのカード文言は短い名詞で統一する

## マスター管理
- 「部位」「種目」「有酸素種別」のタブ切替
- 初期値を用意し、管理者が編集可能
- 有酸素種別は管理者が編集可能（追加/削除/名称変更）
- 一覧はリスト表示（名称 + 編集/削除アイコン）
- 追加は一覧上部の入力＋追加ボタン
- 編集はインライン編集
- 削除は確認ダイアログを表示する
- タブのラベルは「部位」「種目」「有酸素種別」
- 追加入力のプレースホルダーは「新しい項目を追加」
- 削除確認ダイアログ文言は「この項目を削除します。よろしいですか？」

## データ出力
- CSV/JSON両方対応
- 期間指定はFrom/Toの日付ピッカー
- 出力形式はラジオボタン（CSV/JSON）で選択
- 出力ボタンの文言は「エクスポート」
- ヘッダーはタイトル＋戻るリンク
- 戻るリンク文言は「戻る」
- 期間指定のラベルは「開始日」「終了日」
- エクスポート後はダウンロード開始（成功トースト表示）
